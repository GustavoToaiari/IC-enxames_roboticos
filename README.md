# IC-enxames_roboticos

Este repositório contém os códigos desenvolvidos na Iniciação Científica para controle do robô Pioneer P3-DX utilizando campos potenciais.

## Scripts

- 'pioneer-CP_obstaculo-V0.py'
Primeira implementação de campos potenciais para ir até um determinado goal desviando de um único obstáculo. Controle pouco estável e com muitas melhorias a serem feitas.

- 'pioneer-CP_obstaculo-tuned-V1.py'
Versão com controle melhorado, mantendo a mesma ideia e estrutura.

- 'pioneer-CP_obstaculo-POO-V2.py'
Controle levemente melhorado em relação ao anterior, o foco maior foi em passar o código para POO (Programação Orientada a Objetos).
